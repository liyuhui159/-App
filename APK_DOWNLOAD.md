# APK 下载与安装说明

本仓库已经配置 GitHub Actions 自动打包 Android APK。

## 下载 APK

1. 打开仓库页面：`liyuhui159/-App`。
2. 点击上方 **Actions**。
3. 打开最新一次 **Build Android APK**。
4. 在页面下方找到 **Artifacts**。
5. 下载 `lingxi-ledger-debug-apk`。
6. 解压后得到 `app-debug.apk`，发送到手机安装。

## 手机安装

1. 把 `app-debug.apk` 传到手机。
2. 点击安装。
3. 如果系统提示“禁止安装未知应用”，到系统设置中允许当前文件管理器或浏览器安装未知应用。

## 开启自动记账权限

### 1. 通知监听权限

用于读取微信、支付宝、银行等 App 的通知文字：

```text
系统设置 > 通知使用权 / 通知访问权限 > 灵犀自动记账服务 > 开启
```

### 2. 无障碍权限

用于辅助识别支付页面、账单页面、弹窗中的金额文字：

```text
系统设置 > 无障碍 > 已下载的应用 / 更多已下载服务 > 灵犀无障碍自动记账 > 开启
```

## 注意

- 本 App 不上传服务器，账目保存在手机本地 SQLite。
- Android 会限制后台权限，通知监听和无障碍都必须由用户手动开启。
- 如果微信、支付宝或银行隐藏通知金额，自动记账无法识别隐藏内容。
- Debug APK 未签正式发布证书，适合自己测试安装；正式发布应用市场前需要生成 Release 签名 APK。
