# 智能自动记账 App

一个免后端、免构建的智能记账 PWA。手机浏览器打开后可以添加到主屏幕，当作 App 使用；电脑浏览器也可以直接运行。

## 功能亮点

- **智能自动记账**：支持粘贴微信、支付宝、银行账单文字，自动识别金额、收支类型、分类、支付账户、日期和备注。
- **批量导入**：一行一笔，批量解析后可一键写入账本。
- **自动分类**：内置餐饮、交通、购物、住房、学习、医疗、娱乐、收入等关键词规则。
- **周期账单**：支持房租、工资、会员、贷款等月度固定账单，点击“执行自动入账”即可补账。
- **预算管理**：按月设置分类预算，自动显示预算进度和超支提醒。
- **统计看板**：显示本月收入、支出、结余、今日支出、分类占比图和近 7 日趋势。
- **交易流水**：支持搜索、筛选、删除、CSV 导出和 CSV 导入。
- **本地隐私**：数据保存在浏览器 localStorage，不上传服务器。
- **PWA 离线使用**：支持安装到桌面/手机主屏幕，离线打开。

## 如何运行

直接打开 `index.html` 即可运行。

也可以使用 VS Code 的 Live Server 插件预览，或开启任意静态服务器：

```bash
python -m http.server 8080
```

然后访问：

```text
http://localhost:8080
```

## 如何部署到 GitHub Pages

1. 打开仓库 Settings。
2. 进入 Pages。
3. Source 选择 `Deploy from a branch`。
4. Branch 选择 `main`，目录选择 `/root`。
5. 保存后等待 GitHub 生成访问地址。

## 后续改成 APK 的方向

当前版本是 PWA。若要打包为真正 Android APK，可以用以下方式继续扩展：

- 使用 Capacitor 将本项目打包为 Android APK。
- 或将前端界面迁移到 Flutter / Kotlin。
- 若要实现真正读取短信、通知、支付宝/微信账单，需要 Android 原生权限和无障碍/通知监听能力，网页 PWA 不能直接读取这些隐私数据。

## 示例账单文本

```text
今天午餐 18.5 微信
支付宝付款 美团外卖 32.8
工资收入 6500 招商银行 2026-06-19
滴滴打车 -24.6
京东购物 199.00 信用卡
```

粘贴到“智能导入”输入框后，点击“智能解析”，即可自动生成账目。
